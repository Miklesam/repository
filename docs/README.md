# GitHub Pages для коллекции игр

Эта папка содержит файлы для GitHub Pages, которые отображают информацию о всех играх в репозитории.

## Настройка GitHub Pages

1. Перейдите в настройки репозитория на GitHub
2. Найдите раздел "Pages" в боковом меню
3. В разделе "Source" выберите:
   - Branch: `main` (или вашу основную ветку)
   - Folder: `/docs`
4. Нажмите "Save"

После этого ваш сайт будет доступен по адресу:
`https://yourusername.github.io/repository/`

## Обновление информации об играх

Чтобы обновить информацию об играх на странице, отредактируйте файл `index.html` и обновите массив `games` в JavaScript коде.

## Структура

- `index.html` - главная страница с информацией о всех играх
- `README.md` - этот файл с инструкциями

## Кастомизация

Вы можете настроить:
- Цвета и стили в секции `<style>`
- Структуру данных игр в массиве `games`
- Добавить больше информации о каждой игре
- Добавить скриншоты или иконки игр
