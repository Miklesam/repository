# Настройка GitHub Pages

## Быстрый старт

1. **Закоммитьте и запушьте изменения:**
   ```bash
   git add docs/
   git commit -m "Add GitHub Pages"
   git push
   ```

2. **Настройте GitHub Pages:**
   - Перейдите в настройки репозитория на GitHub
   - Найдите раздел "Pages" в боковом меню
   - В разделе "Source" выберите:
     - Branch: `main` (или вашу основную ветку)
     - Folder: `/docs`
   - Нажмите "Save"

3. **Подождите несколько минут** - GitHub обработает изменения

4. **Откройте ваш сайт:**
   - Адрес будет: `https://yourusername.github.io/repository/`
   - Замените `yourusername` на ваш GitHub username

## Обновление информации об играх

Чтобы добавить или обновить информацию об играх:

1. Откройте файл `docs/index.html`
2. Найдите массив `games` в JavaScript коде
3. Обновите данные о нужной игре
4. Закоммитьте и запушьте изменения

## Кастомизация

Вы можете настроить:
- **Цвета**: измените значения в секции `<style>` (градиент фона, цвета карточек)
- **Структуру**: добавьте новые поля в объекты игр (например, скриншоты, ссылки на APK)
- **Стиль**: измените CSS для карточек игр

## Структура файлов

```
docs/
├── index.html      # Главная страница
└── README.md       # Документация для GitHub Pages
```

## Примечания

- GitHub Pages автоматически обновляется при каждом push в ветку `main`
- Изменения могут появиться с задержкой до 10 минут
- Убедитесь, что папка `docs/` не игнорируется в `.gitignore`
