# Tower Defense

Классическая игра типа tower defense для Android, написанная на Kotlin. Защищайте базу от волн врагов, размещая башни вдоль пути.

## Основные особенности

- **Система пути** - враги движутся по заданному S-образному пути
- **Типы башен**:
  - **Базовая** (50 золота) - сбалансированная башня
  - **Быстрая** (75 золота) - высокая скорость стрельбы
  - **Снайпер** (100 золота) - большой урон и дальность
- **Типы врагов**:
  - **Базовый** - стандартный враг
  - **Быстрый** - высокая скорость, меньше здоровья
  - **Танк** - много здоровья, медленный
- **Система волн** - каждая волна становится сложнее
- **Экономика** - зарабатывайте золото за убийство врагов
- **Жизни** - теряйте жизни, когда враги доходят до конца пути

## Управление

- **Касание экрана** - разместить башню (пока только базовая)
- **Кнопка "Начать волну"** - запустить следующую волну врагов
- **Кнопка "Пауза"** - приостановить игру

## Структура модуля

- `MainActivity` - главная активность с UI
- `TowerDefenseView` - основной игровой view с логикой
- `model/Path` - система пути для врагов
- `model/Tower` - башни с системой стрельбы
- `model/Enemy` - враги с движением по пути
- `model/Bullet` - пули башен
- `model/WaveManager` - управление волнами

## Сборка и запуск

Модуль подключён в `settings.gradle` как `games:4`. Для сборки:

```bash
./gradlew :games:5:assembleDebug
```

APK можно установить на устройство или эмулятор с Android API не ниже минимальной, определённой в общем `libs.versions.toml`.

## Игровая механика

- Игра запускается в альбомной ориентации
- Башни автоматически находят и стреляют по ближайшим врагам в радиусе
- Враги движутся по пути с разной скоростью в зависимости от типа
- За каждого убитого врага вы получаете золото
- Игра заканчивается, когда жизни заканчиваются или вы выигрываете все волны
