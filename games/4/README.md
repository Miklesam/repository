# Sky Turret

Казуальная аркада для Android, написанная на Kotlin. Игрок управляет турелью у нижней границы экрана, смещая её пальцем и автоматически ведя огонь по противникам, которые спускаются сверху. Пропущенный враг отнимает жизнь, столкновение с турелью — тоже. Игра завершается, когда заканчиваются три жизни.

## Основные особенности

- **Кастомное игровое поле** `ShooterGameView` с ручной отрисовкой и игровой логикой
- **Простое управление**: перемещение турели касанием экрана
- **Прогрессия сложности**: уровень сложности увеличивается каждые 15 секунд
  - Увеличение скорости врагов
  - Уменьшение интервала появления врагов
  - Появление более сложных типов врагов на высоких уровнях
- **Система комбо**: последовательные попадания увеличивают множитель очков (до x5)
- **Разные типы врагов**:
  - **Обычные** — стандартные враги
  - **Маленькие** — меньше размером, дают меньше очков
  - **Большие** — медленнее, но дают больше очков
  - **Быстрые** — появляются с 5 уровня, двигаются быстрее
- **Визуальные эффекты**: взрывы при попадании с анимацией частиц
- **Адаптивная скорость** движения с учётом DPI устройства
- **Поддержка возобновления** игры после сворачивания приложения

## Структура модуля

- `build.gradle` — настройка Android-приложения с view binding.
- `MainActivity` — стартовая активити с UI управления.
- `ShooterGameView` — основной игровой компонент.
- `res/` — макеты, темы и ресурсы интерфейса.

## Сборка и запуск

Модуль подключён в `settings.gradle` как `games:4`. Со сборкой Gradle проблем быть не должно:

```bash
./gradlew :games:4:assembleDebug
```

APK можно установить на устройство или эмулятор с Android API не ниже минимальной, определённой в общем `libs.versions.toml`.
