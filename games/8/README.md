# Brick Breaker

Классическая игра "Разбей кирпичи" (Арканоид) для Android, написанная на Kotlin. Управляйте платформой, отбивайте мяч и разбивайте все блоки!

## Основные особенности

- **Классическая механика** - управление платформой, отскок мяча, разрушение блоков
- **Физика отскоков** - реалистичные отскоки от платформы и стен
- **Система уровней** - каждый уровень сложнее предыдущего
- **Многоуровневые блоки** - некоторые блоки требуют несколько ударов
- **Визуальные эффекты** - красивые частицы при разрушении блоков
- **Прогрессия сложности** - больше блоков и больше жизней требуется на высоких уровнях

## Управление

- **Касание экрана** - управление платформой (следует за пальцем)
- **Запуск мяча** - мяч запускается при первом касании
- **Автоматическое движение** - мяч автоматически отскакивает и движется

## Структура модуля

- `MainActivity` - главная активность с UI
- `BrickBreakerView` - основной игровой view с логикой
  - Физика мяча и отскоков
  - Система блоков с разными уровнями прочности
  - Система частиц для эффектов
  - Генерация уровней

## Сборка и запуск

Модуль подключён в `settings.gradle` как `games:8`. Для сборки:

```bash
./gradlew :games:8:assembleDebug
```

APK можно установить на устройство или эмулятор с Android API не ниже минимальной, определённой в общем `libs.versions.toml`.

## Игровая механика

- Игра запускается в портретной ориентации
- Платформа управляется касанием экрана
- Мяч запускается при первом касании и автоматически отскакивает
- Блоки расположены вверху экрана в несколько рядов
- Разные блоки требуют разное количество ударов для разрушения
- Мяч отскакивает от платформы под углом в зависимости от точки удара
- Потеря мяча уменьшает количество жизней
- Разрушение всех блоков завершает уровень
- Каждый новый уровень сложнее предыдущего

## Особенности реализации

- Реалистичная физика отскоков с учетом угла удара
- Система частиц для визуальных эффектов
- Оптимизированная отрисовка с использованием Canvas
- Адаптивная генерация уровней
- Плавная анимация и движение объектов
