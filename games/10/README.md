# Basketball Shooter

Баскетбольная игра для Android, написанная на Kotlin. Натягивайте мяч и забрасывайте его в корзину!

## Основные особенности

- **Механика натягивания** - натягивайте мяч для определения силы и направления броска
- **Физика полета** - реалистичная траектория мяча с гравитацией
- **Визуальная индикация силы** - цветная линия показывает силу броска
- **Вращение мяча** - мяч вращается в полете
- **Таймер** - игра на время (60 секунд)
- **Статистика** - отслеживание попаданий и общего количества бросков
- **Красивая графика** - детализированная корзина с сеткой и щитом

## Управление

- **Натягивание мяча** - касайтесь и тяните мяч для настройки броска
- **Отпускание** - отпустите палец для броска
- **Автоматический сброс** - мяч автоматически возвращается после броска

## Структура модуля

- `MainActivity` - главная активность с UI
- `BasketballView` - основной игровой view с логикой
  - Физика броска и полета мяча
  - Система натягивания для определения силы
  - Проверка попадания в корзину
  - Таймер и система очков
  - Визуализация траектории

## Сборка и запуск

Модуль подключён в `settings.gradle` как `games:10`. Для сборки:

```bash
./gradlew :games:10:assembleDebug
```

APK можно установить на устройство или эмулятор с Android API не ниже минимальной, определённой в общем `libs.versions.toml`.

## Игровая механика

- Игра запускается в портретной ориентации
- Мяч находится в нижней части экрана
- Натягивайте мяч для определения силы и направления броска
- Длина линии натяжения определяет силу броска
- Цвет линии меняется от зеленого (слабо) до красного (сильно)
- Мяч летит по параболической траектории с гравитацией
- Попадание в корзину дает очки
- Игра длится 60 секунд
- Отслеживается количество попаданий и общее количество бросков

## Особенности реализации

- Реалистичная физика с гравитацией
- Система натягивания для интуитивного управления
- Визуальная обратная связь через цветную линию
- Вращение мяча в полете
- Детализированная отрисовка корзины и сетки
- Оптимизированная отрисовка с использованием Canvas
