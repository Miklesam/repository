# Space Runner

Космический раннер для Android, написанный на Kotlin. Управляйте космическим кораблем, уклоняясь от препятствий и собирая звезды!

## Основные особенности

- **Управление акселерометром** - наклоняйте устройство влево/вправо для управления кораблем
- **Динамическая скорость** - скорость увеличивается со временем, делая игру все сложнее
- **Типы препятствий**:
  - **Маленькие** - маленькие астероиды, быстрые и маневренные
  - **Средние** - средние астероиды, стандартная сложность
  - **Большие** - большие астероиды, медленные но опасные
- **Система звезд** - собирайте звезды для получения дополнительных очков
- **Визуальные эффекты** - красивые частицы при взрывах и сборе звезд
- **Прогрессия сложности** - чем дальше вы летите, тем быстрее становится игра

## Управление

- **Наклон устройства** - управление кораблем влево/вправо
- **Автоматическое движение** - корабль автоматически летит вперед
- **Сбор звезд** - просто пролетите через звезду, чтобы собрать её

## Структура модуля

- `MainActivity` - главная активность с UI
- `SpaceRunnerView` - основной игровой view с логикой
  - Управление через акселерометр
  - Система препятствий и звезд
  - Система частиц для эффектов
  - Прогрессия скорости и сложности

## Сборка и запуск

Модуль подключён в `settings.gradle` как `games:7`. Для сборки:

```bash
./gradlew :games:7:assembleDebug
```

APK можно установить на устройство или эмулятор с Android API не ниже минимальной, определённой в общем `libs.versions.toml`.

## Игровая механика

- Игра запускается в портретной ориентации
- Корабль автоматически движется вперед с увеличивающейся скоростью
- Препятствия появляются сверху и движутся вниз
- Звезды появляются периодически для сбора
- Столкновение с препятствием заканчивает игру
- Сбор звезд дает дополнительные очки
- Скорость увеличивается каждую секунду, делая игру все сложнее
- Дистанция отслеживается и отображается в реальном времени

## Особенности реализации

- Использование акселерометра для плавного управления
- Система частиц для визуальных эффектов
- Оптимизированная отрисовка с использованием Canvas
- Адаптивная сложность в зависимости от скорости
- Плавная анимация и вращение объектов
