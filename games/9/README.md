# Platform Jumper

Платформер с прыжками для Android, написанный на Kotlin. Управляйте персонажем, прыгайте по платформам, собирайте монеты и избегайте шипов!

## Основные особенности

- **Механика прыжков** - касание экрана для прыжка
- **Гравитация и физика** - реалистичное движение с гравитацией
- **Генерация платформ** - платформы генерируются по мере продвижения
- **Система камеры** - камера следует за игроком
- **Сбор монет** - собирайте монеты для получения очков
- **Препятствия** - избегайте шипов на платформах
- **Визуальные эффекты** - красивые частицы при сборе монет

## Управление

- **Касание экрана** - прыжок (только когда персонаж на земле)
- **Автоматическое движение** - персонаж автоматически движется вперед
- **Гравитация** - персонаж падает под действием гравитации

## Структура модуля

- `MainActivity` - главная активность с UI
- `PlatformerView` - основной игровой view с логикой
  - Физика прыжков и гравитации
  - Система платформ с генерацией
  - Система монет и препятствий
  - Система камеры
  - Система частиц для эффектов

## Сборка и запуск

Модуль подключён в `settings.gradle` как `games:9`. Для сборки:

```bash
./gradlew :games:9:assembleDebug
```

APK можно установить на устройство или эмулятор с Android API не ниже минимальной, определённой в общем `libs.versions.toml`.

## Игровая механика

- Игра запускается в портретной ориентации
- Персонаж стоит на платформах и может прыгать касанием экрана
- Гравитация тянет персонажа вниз
- Платформы генерируются автоматически по мере продвижения
- Камера следует за игроком, создавая эффект бесконечного бега
- Монеты появляются на платформах для сбора
- Шипы на платформах убивают игрока при касании
- Игра заканчивается при падении вниз или столкновении с шипом
- Очки начисляются за собранные монеты
- Дистанция отслеживается и увеличивается по мере продвижения

## Особенности реализации

- Реалистичная физика с гравитацией и прыжками
- Система камеры, следующей за игроком
- Процедурная генерация платформ
- Оптимизированная отрисовка с использованием Canvas
- Система частиц для визуальных эффектов
- Плавная анимация и движение объектов
